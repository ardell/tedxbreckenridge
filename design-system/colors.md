### Tint Usage

Each theme color may be used at 20%, 40%, 60%, or 80% opacity for backgrounds, accents, and visual hierarchy.

### Application Guidelines

- Use theme colors for event-specific materials (fliers, social media, website)
- Always maintain TED Red for logo and primary brand elements
- Create visual hierarchy by varying opacity and color combinations
- Ensure sufficient contrast for readability

---

## Color Usage Best Practices

### Accessibility
- Maintain minimum contrast ratio of 4.5:1 for body text
- Use darker theme colors (Twilight, Eclipse) for text on light backgrounds
- Test all color combinations for readability

### Consistency
- Document color choices in design files
- Reference this guide when creating new materials
- Keep brand colors consistent across all touchpoints

### Implementation

The website's CSS color scheme is defined in `website/assets/css/main.css`. See the "Inversion" theme section for current implementation.

For future years, update this document with new theme colors while maintaining the permanent brand colors.
